# Java Grade Calculator

This is a simple Java program that:

- Takes user input (name, age, score)
- Determines if the user is an adult or minor
- Calculates grade based on score
- Uses loops, conditions, and Scanner

## Features
- Input validation
- Grade system (A, B, C, F)
- Interactive console program

- Java
